Key Financial Calculations

Return on Equity (ROE)

$$ROE = \frac{\text{Net Income}}{\text{Shareholder Equity}} = \frac{1.2B}{8.5B} = 14.1\%$$

Compound Annual Growth Rate (CAGR)

$$CAGR = \left(\frac{V_f}{V_i}\right)^{\frac{1}{n}} - 1$$

$$CAGR_{3yr} = \left(\frac{4.8B}{3.2B}\right)^{\frac{1}{3}} - 1 = 14.5\%$$

Sharpe Ratio

$$S = \frac{R_p - R_f}{\sigma_p}$$

Where $R_p$ is portfolio return, $R_f$ is risk-free rate, and $\sigma_p$ is standard deviation

These formulas demonstrate our strong financial health. The ROE of 14.1% is above industry average of 12%. Our 3-year CAGR of 14.5% shows consistent growth trajectory. The Sharpe ratio will be discussed in detail in the portfolio performance section.

Transaction Analysis Query

Daily transaction monitoring for fraud detection:


-- Identify suspicious transaction patterns
SELECT 
    account_id,
    COUNT(*) as transaction_count,
    SUM(amount) as total_amount,
    AVG(amount) as avg_amount,
    STDDEV(amount) as amount_stddev
FROM transactions
WHERE 
    transaction_date >= CURRENT_DATE - INTERVAL '7 days'
    AND status = 'completed'
GROUP BY account_id
HAVING 
    COUNT(*) > 100 
    OR SUM(amount) > 1000000
    OR STDDEV(amount) > AVG(amount) * 3
ORDER BY total_amount DESC;

This query helped identify 2,347 potentially fraudulent transactions in Q4

Our fraud detection system runs this query every 15 minutes. It flags accounts with unusual transaction patterns for manual review. The system has a false positive rate of only 0.8% and has saved an estimated $47M in prevented fraud this quarter.

// Real-time portfolio value calculator
class PortfolioTracker {
    constructor(holdings) {
        this.holdings = holdings;
        this.websocket = new WebSocket('wss://market-data.api');
    }
    
    calculateTotalValue() {
        return this.holdings.reduce((total, holding) => {
            const currentPrice = this.getMarketPrice(holding.symbol);
            return total + (holding.quantity * currentPrice);
        }, 0);
    }
    
    getPerformanceMetrics() {
        return {
            totalValue: this.calculateTotalValue(),
            dayChange: this.calculateDayChange(),
            totalReturn: this.calculateTotalReturn(),
            sharpeRatio: this.calculateSharpeRatio()
        };
    }
}
